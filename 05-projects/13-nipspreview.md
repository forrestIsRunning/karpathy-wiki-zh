# NIPS Preview

**Website:** https://cs.stanford.edu/people/karpathy/nipspreview/
**GitHub:** https://github.com/karpathy/nipspreview
**Stars:** 186 | **Forks:** 65 | **Language:** Python

## Description

NIPS 2012 accepted papers in a nicer, more readable format. Scripts generate .html to more easily browse NIPS papers.

For each paper, the page shows:
- Title and authors
- Links to PDF, BibTeX, and supplementary materials
- A link to rank by tf-idf similarity to this paper
- Abstract (expandable)
- TOP 100 most-occurring words in the paper, color-coded by LDA topic model with k=7 topics:
  - Topic 0: Theory
  - Topic 1: Reinforcement Learning
  - Topic 2: Graphical Models
  - Topic 3: Deep Learning/Vision
  - Topic 4: Optimization
  - Topic 5: Neuroscience
  - Topic 6: Embeddings etc.

Users can toggle sorting by individual LDA topics to filter papers by subject area.
