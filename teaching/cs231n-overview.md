# CS231n: 卷积神经网络与视觉识别

> 斯坦福大学第一门深度学习课程。Karpathy 设计并主讲。

## 三岁版

这是一门教电脑"看懂"图片的课程。就像教小孩认字一样——先看很多很多图片，然后学会认出"猫"、"狗"、"汽车"。

**为什么这么重要？** 这是世界上第一门系统的深度学习课程，影响了无数后来者。

## 课程规模

- **2015 年**：150 名学生
- **2016 年**：330 名学生（Karpathy 主讲）
- **2017 年**：750 名学生
- 成为斯坦福历史上**最大**的课程之一

## 课程内容

1. **图像分类** — 给一张图，问 AI"这是什么？"
2. **神经网络基础** — 神经元、层、激活函数
3. **卷积神经网络（CNN）** — 专门处理图片的网络结构
4. **训练技巧** — 怎么让网络学得又快又好
5. **计算机视觉应用** — 检测、分割、视频理解
6. **生成模型** — 让 AI 自己画画

## 课程作业

1. **KNN 分类器** — 最简单的图像分类方法（从零实现）
2. **SVM / Softmax 分类器** — 线性分类器
3. **两层神经网络** — 从零实现
4. **卷积神经网络** — 用 PyTorch 训练 CNN
5. **迁移学习** — 用预训练模型做自己的任务

## 历史意义

CS231n 改变了深度学习的教育方式：

1. **首次**把深度学习作为一门独立的本科/研究生课程
2. **Jupyter Notebook + 作业驱动**的教学方式后来被广泛复制
3. 课程笔记 CS231n GitHub Pages 成为全球最流行的深度学习参考资料

## 三句话总结

1. 这门课教电脑用"神经网络"来看懂图片
2. 它是斯坦福第一门深度学习课，从150人涨到750人
3. 课程笔记和视频至今仍是学习 CV 最好的免费资源之一
