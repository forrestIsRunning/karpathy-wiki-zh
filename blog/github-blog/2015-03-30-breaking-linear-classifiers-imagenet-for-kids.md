# 骗电脑的图片 -- 

电脑看图片已经很厉害了。你给它一张熊猫的照片，它能认出来："这是熊猫！"

但是科学家发现了一件奇怪的事：他们可以给照片加上一点点、你眼睛完全看不出来的小变化，电脑就会突然说："这是长臂猿！"或者："这是校车！"

这是怎么回事？

Karpathy 叔叔解释说：这不是电脑的"bug"，也不是深度学习的问题。问题出在——电脑是拿数学算的，数学里有很多小陷阱。

打个比方：你画了一只猫，然后在纸上用铅笔轻轻加了几根几乎看不见的线，你妈妈还是觉得是猫，但是你的朋友突然说"这是狗！"。电脑就像这个"容易上当的朋友"。

这件事告诉我们：电脑虽然厉害，但它看世界的方式和人类完全不一样。你眼睛觉得没区别，电脑可能已经觉得"完全不一样了"！

所以要小心——不要随便相信电脑说"我很确定"哦！
