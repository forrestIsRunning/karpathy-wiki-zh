# 电脑看不懂照片里的笑话 -- 

有一张照片特别好笑：奥巴马把脚踩在别人量体重的秤上。

你看了一眼就笑了，因为你知道：
- 那是奥巴马
- 他在偷偷踩秤
- 那个秤会显示错误的数字
- 量体重的人会吓一跳："我怎么变胖了？"
- 奥巴马在恶作剧！

但是电脑看这张照片，只能看到一堆花花绿绿的小点点。它不知道什么是"秤"、什么是"开玩笑"、什么是"恶作剧"。

Karpathy 叔叔说：电脑要想真正理解一张照片，需要知道好多好多事情——物理怎么工作、人们怎么想、什么好笑什么不好笑。我们现在离这一天还非常非常远。

所以下次你看到一个好笑的表情包，要记得：你能笑出来，已经比最聪明的电脑还要厉害了！
