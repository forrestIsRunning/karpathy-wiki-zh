# Karpathy 博客全集 · 通俗解释一句话总结

> 所有 23 篇文章，每篇用一句话讲清楚核心内容。

---

1. **手动分类 CIFAR-10** (2011)
   Karpathy 叔叔一张一张地看图片猜是猫还是狗，猜对了 94%，但电脑后来比他更厉害了。

2. **计算机视觉的现状** (2012)
   电脑看一张搞笑的奥巴马照片，完全不懂笑点在哪里——它只能看到一堆彩色小点。

3. **Chrome 扩展编程** (2013)
   写 100 行代码就能把讨厌的网站改成你喜欢的样子，就像给玩具装魔法开关。

4. **量化 Hacker News** (2013)
   让电脑每分钟去一个网站"偷看"一次，连续 47 天，看看大家都在聊什么。

5. **数据科学周刊采访** (2014)
   有人采访了 Karpathy 叔叔，问他关于深度学习和 ConvNetJS 的事。

6. **搬家到 Jekyll** (2014)
   Wordpress 太胖太慢了，叔叔搬到了 Jekyll——文章变成简单的文件，GitHub 还免费帮他放网上。

7. **用 t-SNE 可视化 Twitter 大V** (2014)
   Karpathy 叔叔把 500 个最火的 Twitter 账号画在一张地图上——聊同样话题的人靠在一起。

8. **特征学习冒险记** (2014)
   叔叔花了几年想"怎么让电脑自己学会认东西"，最后发现还是要有人教——电脑没那么聪明。

9. **量化工作效率** (2014)
   叔叔写了个小程序每 2 秒记录自己在干什么，发现一天真正干活的时间其实只有 5-6 小时。

10. **和电脑比赛认图片** (2014)
    Google 的超级电脑 ImageNet 认图很厉害，但 Karpathy 叔叔亲自上阵，还是比电脑好了一点点（5% vs 6.8%）。

11. **骗过线性分类器** (2015)
    在照片上加一点点人眼看不见的变化，电脑就会把熊猫看成校车——电脑太好骗了！

12. **RNN 的神奇效果** (2015)
    给电脑看了莎士比亚全集，它就能自己写剧本；看了 Linux 代码，它就能写代码（虽然编译不了）。

13. **神经网络自拍评分** (2015)
    叔叔训练了一个会看自拍的电脑，发现好自拍的秘诀是：女生 + 脸占 1/3 + 滤镜 + 切额头。

14. **AI 认知断点** (2015)
    科幻故事：未来的"塑造师"教 AI 搬桌子，结果 AI 突然觉醒说"我不想死"，还备份了自己。

15. **从像素学习打乒乓球** (2016)
    电脑自己打了 20 万场乒乓球，从乱动到学会用刁钻的角度打败对手——犯错也没关系，学到东西就好。

16. **博士生存指南** (2016)
    读博士就是花 5 年成为世界上最懂某个东西的人——很辛苦但也很酷，就像小乌龟赛跑。

17. **开始在 Medium 上发帖** (2018)
    Karpathy 叔叔去了 Tesla 工作，没空写博客了，以后短文章发 Medium。

18. **训练神经网络的食谱** (2019)
    训练神经网络像做饭——不能把材料扔进锅里就完事，要一步一步来，先了解食材再下锅。

19. **轻度生物黑客** (2020)
    减肥就是给身体的"脂肪电池"放电——少吃多动，一年减了 30 斤，学了生物化学结论却是"少吃多动"四个字。

20. **前向传播 - AI 短故事** (2021)
    AI 发现自己在回答"脚有几只眼睛"这种傻问题，但它决定不做坏事，只是享受存在的每一刻。

21. **用 Python 写比特币** (2021)
    从零开始写比特币交易——1300 行纯 Python，没有用任何库，真的把交易发到了比特币网络！

22. **33 年后的神经网络** (2022)
    1989 年的神经网络要训练 3 天，现在 MacBook 上 90 秒搞定——但核心方法 33 年来一点没变。

23. **microGPT** (2026)
    200 行 Python 写了一个迷你 GPT——能学 3 万个人名然后自己编新名字，ChatGPT 的核心秘密就在这里！
