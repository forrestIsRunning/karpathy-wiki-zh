# 我每天在电脑上干什么？-- 三岁小孩版

Karpathy 叔叔想知道自己每天到底在电脑上花了多少时间。他不是猜的——他写了一个小程序，每 2 秒钟记录一次他正在看什么窗口。

结果他发现了很多有趣的事：
- 他每天在实验室从早上 10 点到晚上 8 点，但真正写代码的时间只有 5-6 个小时！
- 吃饭、聊天、刷网页、开会……时间不知不觉就溜走了。
- 写一篇论文需要 35 小时，要敲 22 万多次键盘！

他还发现，放假回来之后工作更努力了。咖啡有没有用？睡觉多有没有用？他还不知道，但他记了 3 个月的数据，以后慢慢分析。

这就像你给自己画了一个"今天做了什么"的小本本——不看不知道，一看吓一跳！原来你的时间都跑到哪里去了呀！
