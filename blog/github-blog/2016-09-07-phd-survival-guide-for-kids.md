# 博士生存指南 -- 

Karpathy 叔叔终于读完博士了！他写了一份指南，告诉想读博士的小朋友应该怎么做。

读博士是什么？就是你花 5 年时间，成为一个世界上最懂某个东西的人。

**要不要读博士？**
- 你想自己决定研究什么吗？博士可以。
- 你想成为全世界最懂某个东西的人吗？博士可以。
- 但是博士也很辛苦——你会在周末别人出去玩的时候还在实验室里写论文，会连着好几个月发现自己的研究方向走错了。

**怎么选导师？**
导师就像你的"研究生爸爸/妈妈"。年轻导师会手把手教你做数学题。资深导师则更多给你大方向的建议。一定要找跟你合得来的！

**怎么写论文？**
Karpathy 叔叔说：一篇论文只讲一个核心的东西。不要想把两个东西塞进一篇论文。

**怎么演讲？**
搞笑点！放很多图片！讲成一个故事！不要密密麻麻的文字！

**最重要的建议：**
做好的研究，好好告诉大家。不要耍小聪明（比如故意多引用自己的文章）。踏踏实实地做事，别人会注意到你的。

就像小乌龟赛跑：慢慢来，但一直走，最后就到了终点！
