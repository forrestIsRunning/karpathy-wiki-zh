# 自己动手写比特币 -- 

Karpathy 叔叔说：区块链很酷，因为它让我们不仅共享代码，还能共享一台在全世界运行的电脑。

为了搞懂比特币是怎么工作的，他决定从零开始用 Python 写一个比特币交易——不使用任何现成的库！

他需要：
1. 生成一个密码身份（就像你的"数字钱包"）
2. 创建公钥和私钥（私钥就是你的密码，谁拿到谁就能花你的钱）
3. 写 SHA256 哈希函数（一种把任何东西变成"指纹"的算法）
4. 生成比特币地址（就像你的银行卡号）
5. 创建交易输入和输出（谁给了谁多少钱）
6. 数字签名（证明这钱是你同意转出去的）

整个过程大概花了 1300 行 Python 代码。而且这一切都在纯 Python 中完成——没有用任何现成的加密库！

最酷的是：Karpathy 叔叔真的创建了一个比特币测试交易，并且广播到了比特币的测试网络。你可以在区块链浏览器上查到它！

所以比特币不是什么神秘的东西——它只是一大堆数学和代码。只要有耐心，你也能自己写出来！
