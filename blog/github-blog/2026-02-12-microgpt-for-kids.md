# 最小的 GPT -- 

Karpathy 叔叔做了一个叫 microGPT 的东西。它只有 200 行 Python 代码，没有用任何外部库，却包含了一个完整 GPT 模型的所有核心部分！

这个 tiny 的 GPT 能做什么？它能学习 32,000 个名字，然后自己"发明"新的名字。比如它学会了"emma""olivia"之后，自己编出了"kamon""karai""jaire"这样的名字。

microGPT 包含了所有东西：
- 数据集（名字列表）
- 分词器（把字母变成数字）
- 自动求导引擎（计算"怎么改参数才能更好"）
- GPT-2 类似的神经网络结构
- Adam 优化器
- 训练循环
- 推理生成

虽然它很小（只有 4192 个参数，而 GPT-4 有几千亿个），但是核心算法一模一样！唯一的区别是规模——大模型用了 GPU、用了更好的分词器、用了更多的数据。

Karpathy 叔叔把它做成了一张海报，可以三折打印出来挂在墙上。他说：如果你懂了 microGPT，你就懂了 GPT 的核心精髓。剩下的都只是"工程优化"。

所以，ChatGPT 的秘密就藏在这 200 行代码里！酷不酷？
