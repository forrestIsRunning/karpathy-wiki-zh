# 自拍达人速成班 -- 

Karpathy 叔叔训练了一个超级电脑，教它分辨"好的自拍"和"烂自拍"。

他下载了 200 万张自拍，然后看哪张点赞多、哪张点赞少，教电脑学会判断。

然后电脑告诉了我们自拍的秘密：

**好自拍 = 女生 + 脸占1/3 + 歪头 + 长发 + 滤镜 + 白边 + 切掉一点额头**

**烂自拍 = 太暗 + 脸太大 + 多人合影**

最搞笑的是：叔叔拿名人照片去测试，电脑觉得 Ellen 在奥斯卡的大合照超级棒（虽然前面说多人合影不好）。Chris Pratt 的肌肉照也得了高分。而 James Franco 露了很多肉，但电脑并不感冒。

这个故事告诉我们：电脑会拍照了，而且它还挺挑剔的！
