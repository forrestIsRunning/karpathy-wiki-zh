# Software 2.0 — 三岁小孩版（最重要的文章！）

Karpathy 叔叔说，编程要分两个时代。

**软件 1.0** 就是普通人理解的编程——一个人在电脑前写一行一行的代码，告诉电脑每一步要做什么。就像你拼乐高的时候看说明书，一步一步地拼。

**软件 2.0** 就完全不一样了！你不用告诉电脑每一步怎么做，你只需要给它看好多好多的例子，然后电脑自己学会怎么做。比如你想让 AI 认出一只猫，你不用写"猫有两只耳朵、四条腿"，你只需要给它看几万张猫的图片，说"这些是猫"，它自己就学会了！Karpathy 叔叔说，现在的程序员的工作变了——不再是一行行写代码，而是收集和整理好数据，让"神经网络"自己学会。就像你不是教小朋友背答案，而是给他做很多练习题，他自己就会了。

这就是为什么这篇文章超级有名——因为它说了一个很重要的道理：未来的编程，不是写代码，而是训练 AI。软件 1.0 在吞噬世界，而 AI（软件 2.0）正在吞噬软件本身！
