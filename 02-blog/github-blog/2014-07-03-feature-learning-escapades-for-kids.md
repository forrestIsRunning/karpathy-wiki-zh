# 怎么让电脑自己学会认东西？-- 三岁小孩版

Karpathy 叔叔花了好多年想一个问题：能不能让电脑自己学会认东西，不用人教？

一开始他想：电脑可以自学啊。就像你生下来没人教你怎么看猫，看着看着就知道了。他把很多很多图片和视频给电脑看，希望电脑自己发现"这是猫"、"这是狗"。

但是试了好几年，都不太成功。他试了 3D 数据（像 Kinect 游戏机那种），也不行。他好难过，觉得自己的博士读不下去了。

后来一个叫 AlexNet 的超级电脑出现了。它不是自己学的——而是有人拿了好多好多已经标好的图片（"这张是猫""这张是狗"）去教它。结果它变得超级厉害，比之前所有方法都好！

Karpathy 叔叔终于明白了一个道理：要让电脑认东西，还是需要有人教它。就像小朋友认动物，也需要爸爸妈妈指着说"这是猫猫"。电脑没那么聪明，不能完全自己学。

不过没关系！先把能教的都教给电脑，以后再说以后的事！
