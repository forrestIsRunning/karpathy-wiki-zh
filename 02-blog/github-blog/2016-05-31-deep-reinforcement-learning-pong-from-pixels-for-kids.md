# 让电脑自己学会打乒乓球 -- 三岁小孩版

想让电脑学会打乒乓球游戏（Pong），你要怎么做？

不是教它规则。而是让它自己玩！Karpathy 叔叔用了一种叫"策略梯度"的方法。

一开始电脑什么都不会——它就像刚出生的小宝宝，随便乱动。它打了一局又一局，输了 200,000 场（没错，20 万场！）。

但是每一次输了，它都学到一点点："啊，刚才那样动让我输了，下次不要那样了。"赢了的时候："哦，这样动让我得分了，下次再来！"

经过好多天的训练，电脑终于学会了打乒乓球！它会等球飞过来，然后以一个刁钻的角度打回去，让对手接不到。

这个故事告诉我们：有时候犯错也没关系，只要你从错误中学到东西，最后就会变得越来越厉害！
