# 人和电脑比赛认图片 -- 三岁小孩版

2014 年，有一个叫 ImageNet 的比赛。电脑们比赛认图片——这张是"金毛狗"、那张是"帆船"。

Google 的超级电脑 GoogLeNet 特别厉害，100 张图只认错 6.7 张。

Karpathy 叔叔想：那人类能认对多少呢？于是他亲自上阵，一张一张地看了 1500 张图片。

结果：
- Karpathy 叔叔认错了 5.1%（比电脑好一点点）
- 电脑容易把小东西看错（比如照片角落里的一个小蚂蚁）
- 电脑怕滤镜（加了滤镜就看不懂了）
- 人分不清狗的品种（有 120 多种狗！太多了！）

叔叔说：人很快就会被电脑超过了。以后比认图片，可能只有最厉害、最有耐心的人才比得过电脑。

就像跑步比赛——人会累，但机器不会！
