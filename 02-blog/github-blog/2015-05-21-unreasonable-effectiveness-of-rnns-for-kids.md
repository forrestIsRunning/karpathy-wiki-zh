# 会写字的魔法电脑 -- 三岁小孩版

Karpathy 叔叔训练了一种叫 RNN 的魔法电脑。这种电脑只要看一大堆文字，就能自己学会写东西！

他给了电脑莎士比亚的全部剧本。电脑学会了写剧本——人物名字、台词、标点符号，都写得有模有样！比如：

"国王：来吧，我要看看你的本事。"
"小丑：先生，我会让你大开眼界的。"

他还给了电脑 Linux 的代码（几亿行 C 语言）。电脑竟然学会了写代码！它会写注释、会引用头文件、会用括号、还会缩进。虽然编译不了，但是看起来真的很像程序员写的！

最神奇的是：电脑还学会了维基百科的格式，知道怎么加链接 `[[ ]]`、怎么做标题、怎么列清单。它甚至"发明"了一个不存在的网址！

这就像你给一个小朋友看了好多好多 Lego 说明书，然后他就能自己搭出新的 Lego 房子了——虽然有时候搭得歪歪扭扭的，但是真的很厉害！
